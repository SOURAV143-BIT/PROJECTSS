##Hospital Care Analysis Project

## Overview
The Hospital Care Analysis Project is a Data Analysis project developed using Python libraries such as Pandas, NumPy, and Matplotlib. The project focuses on analyzing hospital patient data to discover useful insights related to patient age, departments, billing, admission details, and hospital stay patterns.

The project includes Exploratory Data Analysis (EDA), data visualization, and statistical insights to better understand healthcare data.



## Features
 Data Cleaning and Preprocessing
 Exploratory Data Analysis (EDA)
 Patient Age Analysis
 Department-wise Analysis
 Billing Analysis
 Hospital Stay Analysis
 Data Visualization using Charts and Graphs



## Technologies Used
 Python
 Pandas
 NumPy
 Matplotlib
 Google Colab



## Dataset Information
The dataset contains hospital-related patient records including:
- Patient ID
- Age
- Gender
- Disease
- Department
- Admission Date
- Billing Amount
- Stay Days
